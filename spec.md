# Environment Spec: [Your Environment Name]

Replace this file with your environment specification. The agent will read this as its north star for what to build.

## What to include

### Task description
What is the agent doing? (e.g., scheduling meetings, writing code, navigating a game, negotiating a deal)

### Tools
What tools does the agent have access to? What do they return?

### Constraints and rules
What rules govern the task? What makes a solution valid or invalid?

### Scoring
How should solutions be scored? What does a perfect score look like? What about partial credit?

### Difficulty levels
What makes a task easy vs. hard? How should the difficulty distribution look?

### Example
Walk through a concrete example task and what a good solution looks like.
